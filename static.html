DASHBOARD

This project was intended to have a static dashboard in which you can adjust your dashboard size. This may helpful if you are just using a phone to open a webpage without ruining its design.

The Step to make a dashboard are:

Step 1: make a html file named as "index.html" and code your texts. And make a css file to design.

Step 2: make a @media in your css file to make it responsive.

Step 3: last step is to test and run your code.
